Project 2: Program a Robot

Link to source code

For this project, I programmed a robot to move depending on the input of the user. Here is a list of what happens for each move:

1. Choose: move in a circle or a rectangle
2. Choose: move forward or backward —> and for how many seconds
3. Choose: sin right or left first —> and for how many seconds
4. Tells you how far the robot is from each wall (because the position of the robot changes depending on the commands you previously chose)
5. Asks you whether you had fun or not!
